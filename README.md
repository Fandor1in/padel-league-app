# Padel League Mini App

This repository contains the source code for a Telegram Mini App that manages a padel league.
It includes a frontend web application (mini-app) built with React/TypeScript and a backend built with Node.js/TypeScript.
The application integrates with Airtable to store league data, and it communicates with users through a Telegram bot.

## Structure

- **frontend/** – contains the code for the Telegram mini-app that runs inside Telegram’s web view.
- **backend/** – contains the server and bot logic, including routes for interacting with Airtable.

For detailed project setup and architecture, refer to the project documentation.
